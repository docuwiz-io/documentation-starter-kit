**👉** Follow these steps in order. Each step builds on the previous one.

### Step 1: Open the **[Module / Feature]**

1. Log in to **[Product Name]**.

1. From the left navigation menu, select **[Module Name]**.

1. The **[Module dashboard/workspace]** appears.

> **Expected result:** ✅ You are on the **[Module]** landing page.

### Step 3: Add your first **[Task / Step / Component]**

1. Select **Add Task / Add Step / Add Component**.

## ⚠️ Common mistakes to avoid

- Skipping mandatory fields

- Not saving before exiting

- Using duplicate names

- Selecting the wrong project/environment

## 🧯 Troubleshooting tips

**▶ ❌ I can’t create a project/workflow**

- Check your permissions

- Verify the module is enabled

**▶ ❌ My changes are not visible**

- Refresh the page

- Clear filters

- Reopen the project

**▶ ❌ Save button is disabled**

- Fill all required fields

- Check for validation errors
